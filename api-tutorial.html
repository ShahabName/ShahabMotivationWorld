üìã API Overview

The ShahabMotivationWorld Mock API is a client-side Service Worker-based REST API designed specifically for QA automation testing practice. It provides a complete set of endpoints for testing authentication flows, data retrieval, and error handling scenarios.

Key Features:

  • ‚úÖ No Backend Required: Runs entirely in the browser using Service Workers
  • ‚úÖ RESTful Design: Follows REST API best practices
  • ‚úÖ Authentication Flow: Token-based authentication with JWT-style tokens
  • ‚úÖ Realistic Responses: Returns properly formatted JSON responses
  • ‚úÖ Error Scenarios: Supports testing various HTTP status codes
  • ‚úÖ Cross-Tool Compatible: Works with all major testing frameworks

Base URL:

https://shahabname.github.io/ShahabMotivationWorld/api

üîó Available Endpoints

Method Endpoint Description Auth Required
GET /status Check API operational status No
GET /token Generate authentication token No
POST /login User login with credentials Yes (Auth Token)
GET /quotes Retrieve all motivational quotes No
GET /getrandomquote Get a random quote No

Important: Tokens persist indefinitely in your browser's IndexedDB. Store the token securely and use it in the Authorization header for protected endpoints. Clear browser data to reset tokens.

3Ô∏è‚É£ User Login

POST /api/login

Authenticate user with credentials and receive a session token.

üì° Request Details

  • Method: POST
  • URL: https://shahabname.github.io/ShahabMotivationWorld/api/login
  • Headers:
    • Authorization: Bearer {authToken}
    • Content-Type: application/json
  • Body: JSON with username and password

Request Body Schema

{ "username": "string (required)", "password": "string (required)" }

üì§ Example Requests

Postman

1. Create new POST request 2. URL: https://shahabname.github.io/ShahabMotivationWorld/api/login 3. Headers: - Authorization: Bearer {{authToken}} - Content-Type: application/json 4. Body (raw JSON): { "username": "demo_user", "password": "password123" } 5. Click "Send"

Playwright API Context

// First get auth token const tokenResponse = await apiContext.get('/api/token'); const authToken = (await tokenResponse.json()).data.token; // Then perform login const loginResponse = await apiContext.post('/api/login', { headers: { 'Authorization': `Bearer ${authToken}`, 'Content-Type': 'application/json' }, data: { username: 'demo_user', password: 'password123' } }); const loginData = await loginResponse.json(); const sessionToken = loginData.data.sessionToken; console.log('Login successful! Session Token:', sessionToken);

REST Assured (Java)

@Test public void testUserLogin() { // First get auth token String authToken = getAuthToken(); // Prepare login credentials Map credentials = new HashMap<>(); credentials.put("username", "demo_user"); credentials.put("password", "password123"); // Perform login Response response = given() .baseUri("https://shahabname.github.io/ShahabMotivationWorld") .header("Authorization", "Bearer " + authToken) .contentType("application/json") .body(credentials) .when() .post("/api/login") .then() .statusCode(200) .body("success", equalTo(true)) .body("data.user.username", equalTo("demo_user")) .extract().response(); String sessionToken = response.jsonPath().getString("data.sessionToken"); System.out.println("Session Token: " + sessionToken); }

Step-by-step example of the full authentication flow from token generation to accessing protected resources.

Complete Playwright Example

import { test, expect } from '@playwright/test'; test.describe('Complete API Authentication Flow', () => { let apiContext; let authToken; let sessionToken; test.beforeAll(async ({ playwright }) => { apiContext = await playwright.request.newContext({ baseURL: 'https://shahabname.github.io/ShahabMotivationWorld' }); }); test('Step 1: Check API Status', async () => { const response = await apiContext.get('/api/status'); expect(response.status()).toBe(200); const data = await response.json(); expect(data.success).toBe(true); expect(data.data.status).toBe('operational'); console.log('‚úì API is operational'); }); test('Step 2: Generate Auth Token', async () => { const response = await apiContext.get('/api/token'); expect(response.status()).toBe(200); const data = await response.json(); authToken = data.data.token; expect(authToken).toBeTruthy(); console.log('‚úì Auth token generated:', authToken); }); test('Step 3: Login with Credentials', async () => { const response = await apiContext.post('/api/login', { headers: { 'Authorization': `Bearer ${authToken}`, 'Content-Type': 'application/json' }, data: { username: 'demo_user', password: 'password123' } }); expect(response.status()).toBe(200); const data = await response.json(); sessionToken = data.data.sessionToken; expect(sessionToken).toBeTruthy(); console.log('‚úì Login successful, session token:', sessionToken); }); test('Step 4: Access Protected Resource', async () => { const response = await apiContext.get('/api/quotes', { headers: { 'Authorization': `Bearer ${sessionToken}` } }); expect(response.status()).toBe(200); const data = await response.json(); expect(data.data.quotes.length).toBeGreaterThan(0); console.log('‚úì Retrieved', data.data.quotes.length, 'quotes'); }); test('Step 5: Get Random Quote', async () => { const response = await apiContext.get('/api/getrandomquote'); expect(response.status()).toBe(200); const data = await response.json(); console.log('‚úì Random quote:', data.data.text); console.log(' Author:', data.data.author); }); test.afterAll(async () => { await apiContext.dispose(); }); });
